## 1. TECHNISCHE SEO AUDIT - BEVINDINGEN

### 1.1 Positieve punten
- Shopify-platform met standaard sitemap.xml en robots.txt
- SSL-certificaat actief (HTTPS)
- BreadcrumbList structured data op alle pagina's aanwezig
- FAQPage schema op /pages/faq en /pages/friendchise (22+ Q&A's)
- WebSite en Organization schema op homepage
- Taal correct ingesteld op Duits (de)
- CDN voor afbeeldingen met breedte-optimalisatie
- Font-optimalisatie via woff2

### 1.2 Kritieke problemen

| Prioriteit | Probleem | Impact |
|---|---|---|
| KRITIEK | Geen meta descriptions op de meeste pagina's | Google toont willekeurige tekstfragmenten, lagere CTR |
| KRITIEK | Meerdere H1-tags op homepage (minstens 3) | Google kan hoofdonderwerp niet bepalen |
| KRITIEK | Geen LocalBusiness schema op studiopagina's | Mist local pack rankings in Google |
| HOOG | Afbeeldingen missen alt-teksten | Slecht voor toegankelijkheid en image search |
| HOOG | Geen Open Graph / Twitter Card meta tags | Slechte social media previews bij delen |
| HOOG | Geen hreflang tags | Niet kritiek nu, maar relevant bij eventuele NL-expansie |
| GEMIDDELD | Blog heeft slechts 3 artikelen | Te weinig content voor organisch verkeer |
| GEMIDDELD | Geen Google Maps embed op studiopagina's | Mist visuele bevestiging voor lokale zoekers |
| GEMIDDELD | Heading tags in lowercase (styling-keuze) | Geen directe SEO-impact maar inconsistent |
| LAAG | Pagina's niet geupdate sinds dec 2025 | Verse content signaleert relevantie aan Google |
